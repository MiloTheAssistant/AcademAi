# AcademAI Brand Asset Usage Copy

Use this as the copy and placement guide for the current AcademAI brand-image library.

## Profile Stamp

Primary file:

```text
public/brand/website/profile-stamp.png
```

Use for:

- Header and footer brand mark
- Social profile pictures
- Browser/app icon source
- Small-format brand recognition

Copy posture:

- AcademAI is the named brand.
- Supporting copy should emphasize guided learning, practice, progress, and proof.
- Do not place provider names inside the stamp or imply official provider certification.

## Website Hero

Primary file:

```text
public/brand/website/profile-background.png
```

Use for:

- Homepage first viewport
- Premium Human Learning Lab positioning
- Broad AI learning and mastery messaging

Suggested copy themes:

- Practical AI mastery
- Guided learning paths
- Scenario practice
- Progress tracking
- Certificates as proof of completion

## Website Section Images

Primary files:

public/brand/website/section-mastery-console.png
public/brand/website/section-assessment-lab.png
public/brand/website/section-syllabus-builder.png
public/brand/website/section-certificate-proof.png

Use for:

- Product sections below the hero

Suggested copy pairings:

- Mastery Console: Track progress, revisit lessons, and keep learning visible.
- Assessment Lab: Practice scenarios, test readiness, and find the next skill gap.
- Syllabus Builder: Generate a focused study path from goals, schedule, and current skill.
- Certificate Proof: Turn completed practice into visible proof of learning.
