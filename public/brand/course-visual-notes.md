# AcademAI Course Visual Notes

This file keeps provider-specific course visuals inside the AcademAI brand system.

## Core Rule

Every course can teach a provider or tool, but the visual system remains AcademAI-first and provider-neutral.

Avoid:

- Official provider logos
- Provider typography mimicry
- Provider UI replicas
- Official-style badges or marks
- Claims that imply provider certification unless formally verified

## Course Visual Lanes

### Beginner Courses

Use Human Learning Lab imagery:

- Hands, notebooks, laptops, and clean study desks
- Soft daylight and calm blue/violet accent glow
- Overlay-safe negative space
- Practical confidence rather than intense technical atmosphere

### Intermediate Courses

Use a Human Learning Lab plus Mastery Console blend:

- Abstract progress arcs
- Scenario cards without readable text
- Study-path map shapes
- Console-like learning cues without fake UI text

### Advanced Or Builder Courses

Use Mastery Console imagery:

- Structured dashboards without readable UI text
- Progress, assessments, generated syllabi, and certificate/proof cues
- More slate contrast with blue/violet highlights
- Practical technical focus, not sci-fi abstraction

## Provider Topic Notes

### Claude / Anthropic Topics

Communicate practical AI fluency, prompt design, model collaboration, and workflow thinking. Do not use Anthropic or Claude logos, marks, typography, or official visual styling.

### OpenAI Topics

Communicate applied AI building, assistants, workflows, and product fluency. Do not use OpenAI logos, marks, typography, or official visual styling.

### Perplexity Topics

Communicate research fluency, answer evaluation, source checking, and knowledge synthesis. Do not use Perplexity logos, marks, typography, or official visual styling.
